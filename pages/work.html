IoT

Projet IoT

Kotlin Jetpack-compose C++ Arduino Firebase

L'IoT (Internet des objets) est un projet qui mesure notre environnement (CO₂, humidité, etc.). Si les conditions deviennent dangereuses pour l’homme, une alarme se déclenche. Le projet dispose également d’une application qui permet de visualiser les données et de nous notifier en cas de danger.
